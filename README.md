# OnlineRPS
An online web app, which allows people to remotely play Rock Paper Scissors with a random or particular person.

## Tools and languages used

* Node JS, MVC Architecture (server creation)
* Web Sockets (making game real time)
* JavaScript (game mechanics creation)
* HTML/CSS, jQuery, JavaScript, w3.css (friendly UI/UX creation)
